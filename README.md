# 🚀 RAG AI Agent with Pinecone & OpenAI

This project demonstrates a complete Retrieval-Augmented Generation (RAG) pipeline that allows users to upload documents and chat with their own data using an AI agent.

---

## 🔥 Features

* 📂 Upload and process documents (Google Drive supported)
* 🧠 Generate embeddings using OpenAI
* 🗄️ Store vectors in Pinecone
* 🤖 AI Agent with memory
* 💬 Chat with your own data

---

## 🏗️ Architecture

1. Data Ingestion (Google Drive / Files)
2. Document Processing
3. Embedding Generation (OpenAI)
4. Vector Storage (Pinecone)
5. AI Agent Query System

---

## 🧰 Tech Stack

* Python
* OpenAI API
* Pinecone Vector Database
* LangChain / Flow Builder
* RAG Architecture

---

## ⚙️ Installation

```bash
git clone https://github.com/your-username/rag-ai-agent-pinecone.git
cd rag-ai-agent-pinecone
pip install -r requirements.txt
```

---

## 🔑 Environment Variables

Create a `.env` file:

```env
OPENAI_API_KEY=your_openai_key
PINECONE_API_KEY=your_pinecone_key
```

---

## ▶️ Usage

```bash
python app.py
```
